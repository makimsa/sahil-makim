# sahil-makim

A React website built with Vite and hosted on GitHub Pages.

## Development

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```
This will start the development server at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Deployment

### Deploy to GitHub Pages
```bash
npm run deploy
```

This will build your project and deploy it to GitHub Pages. Your site will be available at:
`https://makimsa.github.io/sahil-makim/`

## GitHub Pages Setup

After running `npm run deploy`, make sure to:
1. Go to your repository settings on GitHub
2. Navigate to Pages section
3. Set the source branch to `gh-pages`

## Project Structure

- `/src` - React source files
- `/public` - Static assets
- `/dist` - Production build output (generated)
